## Context

Сцены регистрируют `act` и `goto` через `SceneContext`; `flushActsToDOM` и `flushGotosToDOM` в `src/ifKit/renderer.ts` сейчас создают только `<button>`. Стили `.btn-act` и `.btn-goto` в `src/style.css` уже задают разный вид. Пользовательский запрос — развести **семантику DOM** для скринридеров, не меняя внешний вид.

## Goals / Non-Goals

**Goals:**

- Рендер `goto` как `<a>`, `act` как `<button type="button">`.
- Сохранить текущий визуальный стиль `btn-goto` / `btn-act` и раскладку в `#scene-gotos` / `#scene-acts`.
- Поведение при клике идентично текущему (тот же `handler`).
- Клавиатура: как минимум Enter; Space — с паритетом к текущим кнопкам (см. решения ниже).

**Non-Goals:**

- Менять API `ctx.goto` / `ctx.act`, историю, автосохранение, фокус-якорь сцены.
- Вводить реальные URL маршрутизации или интеграцию с `history.pushState`.
- Добавлять обязательные ARIA-атрибуты поверх нативной семантики `button` / `a`.

## Decisions

1. **`href` для `<a>`** — использовать `href="#"` (или другой безопасный in-page placeholder) и в обработчике активации вызывать `preventDefault()` (и при необходимости `stopPropagation()`), чтобы не уводить документ и не прокручивать к якорю непредсказуемо. Скринридер по-прежнему распознаёт элемент как ссылку; видимый текст остаётся `label`.

2. **Space на ссылке** — нативно основная активация ссылки — Enter; Space на фокусированной `<a href="#">` ведёт себя не как у `<button>`. Для паритета с текущим UX добавить обработчик `keydown`: при `Space` и отсутствии модификаторов — `preventDefault()` и вызов того же `handler`, что и для `click`.

3. **Стили** — оставить класс `btn-goto` на `<a>`; в CSS добавить только то, что нужно для сброса дефолтов ссылки (`text-decoration: none`, при необходимости `color` наследование уже перекрыто, `display`/`box-sizing` если flex-ряд ведёт себя иначе), не меняя палитру и размеры из темы.

4. **`type` и безопасность** — для `act` явно оставить `type="button"`. Для `<a>` не использовать `javascript:` URL.

## Risks / Trade-offs

- **[Риск] `href="#"` и объявление ссылки** — технология может озвучивать «ссылка» без понятного назначения URL. **Смягчение:** осознанный trade-off; видимый текст перехода остаётся основной подписью; при отзывах можно добавить опциональный `aria-label` в отдельном изменении.

- **[Риск] Space по умолчанию на странице** — без `preventDefault` возможен скролл. **Смягчение:** обработчик Space с `preventDefault` при фокусе на goto.

- **[Риск] тесты завязаны на `button`** — если появятся — обновить селекторы. **Смягчение:** проверить репозиторий на e2e/юнит DOM.
