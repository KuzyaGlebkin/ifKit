## Why

Интерактивный бренд в `#controls` сейчас вынесен из семантического списка глобальных контролов и оформлен как `<button>`, хотя по смыслу это переход на другой экран (стартовое меню), а не команда в текущем контексте. Вспомогательным технологиям полезнее роль ссылки и единая группировка с остальной панелью, без расхождения «в списке четыре пункта, а фокусируемых элементов пять».

## What Changes

- Перенести интерактивный бренд **внутрь** того же `<ul class="ifk-controls-actions">`, что и четыре глобальные кнопки: первый `<li>` — бренд, далее четыре `<li>` с `#btn-undo` … `#btn-settings` без смены `id` кнопок.
- Заменить разметку бренда с `<button type="button">` на нативную **ссылку** `<a>` (навигационная семантика для AT), сохранив прежний внешний вид (классы логотипа, отсутствие «голубого подчёркивания» как у типичной гиперссылки).
- Обновить стили `#controls`: раскладка «бренд слева, четыре кнопки у правого края» через flex у списка (например отступ у первого `li`), без изменения видимого порядка и акцентного цвета логотипа.
- Обновить обработчик в `session-main-menu.ts`: вешать на элемент бренда (тот же стабильный `id`, ожидаемо `#btn-brand`), предотвращать переход по `href`, сохранить сценарий автосохранения, остановки музыки и показа меню.
- Локализуемое имя для AT (как сейчас через `u(UI.sessionMenuBrand)` или согласованный ключ) перенести на ссылку (`aria-label` или видимый/скрытый текст по правилам проекта).

### Modified Capabilities

- `app-branding`: уточнить требование к интерактивному логотипу (ссылка вместо кнопки, место в разметке относительно списка).
- `ui-buttons`: расширить требование «семантический список» — один `<ul>` с пятью `<li>` (бренд-ссылка + четыре кнопки), порядок Tab и отсутствие ловушек фокуса на контейнерах.
- `project-scaffold`: скелет `index.html` — структура `#controls` с брендом внутри общего списка.
- `session-main-menu`: сценарии клика/фокуса по бренду в навбаре согласовать с элементом-ссылкой и доступным именем.

## Impact

- `src/index.html` — разметка навбара.
- `src/style.css` — селекторы `#btn-brand` / `.ifk-brand-btn` для `<a>`, flex-раскладка `.ifk-controls-actions`.
- `src/ifKit/session-main-menu.ts` — слушатель, i18n, клонирование SVG для меню (селектор к логотипу).
- Возможные упоминания структуры `#controls` в тестах/доках — по факту поиска.
