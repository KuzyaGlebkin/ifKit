## Context

В `src/index.html` бренд — отдельный `<button id="btn-brand">` перед `<ul class="ifk-controls-actions">` с четырьмя кнопками. Стили прижимают список к правому краю (`margin-inline-start: auto` на `ul`). Поведение «в главное меню» реализовано в `session-main-menu.ts` (клик, `aria-label` из `u(UI.sessionMenuBrand)`).

## Goals / Non-Goals

**Goals:**

- Один семантический список действий верхней панели: первая позиция — бренд как **навигационная ссылка**, далее четыре кнопки без смены их `id`.
- Визуально сохранить текущий вид: логотип слева, группа из четырёх кнопок справа в одной строке при достаточной ширине; акцентный цвет; скрытие логотипа на очень узком viewport по существующему правилу.
- Сохранить сценарий: автосохранение, остановка музыки для меню, показ стартового меню; не ломать клонирование SVG для меню из разметки навбара.

**Non-Goals:**

- Смена URL или настоящая навигация документа (приложение остаётся SPA на одной странице).
- Переименование `id="btn-brand"` (оставляем для минимального диффа в TS/CSS; при желании позже можно отдельно переименовать в `link-brand`).

## Decisions

1. **Элемент: `<a>` с `href="#"`**  
   Нативная ссылка даёт роль «link» в AT и ожидаемую озвучку навигации. Обработчик `click` вызывает `preventDefault()` и существующую логику меню, чтобы не менять положение прокрутки и не уходить с страницы.  
   *Альтернатива:* `<button>` + `role="link"` — хуже соответствует запросу «как ссылка».  
   *Альтернатива:* только `href` на якорь меню — лишняя связь с прокруткой и фокусом без выигрыша.

2. **Структура: один `<ul>`, пять `<li>`**  
   Первый `li` содержит ссылку бренда; остальные четыре — прежние кнопки. Раскладка: flex на `ul`, у первого `li` — `margin-inline-end: auto` (или эквивалент), чтобы визуально оттолкнуть четыре кнопки вправо, как сейчас отдельный логотип + список.

3. **Стили**  
   Перенести/дублировать правила с `#btn-brand.ifk-brand-btn` на якорь: сброс подчёркивания, `cursor: pointer`, прозрачный фон, отступы как у кнопки; при необходимости общий класс `.ifk-brand-btn` оставить как «хром бренда», не подразумевая тег `<button>`.

4. **Клавиатура**  
   Для `<a href="#">` активация по Enter — нативная; Space для ссылок не стандартна: при необходимости добавить обработчик `keydown` для Space, зеркалирующий клик (как для кастомных контролов), чтобы паритет с прежней кнопкой не ухудшился.

## Risks / Trade-offs

- **[Risk]** Некоторые пользователи ожидают от ссылки смены URL или новой страницы.  
  **Mitigation:** доступное имя вроде «Главное меню» / локализованный эквивалент явно описывает цель; поведение совпадает с прежним.

- **[Risk]** `href="#"` может дать мелкий сдвиг фокуса/истории в краевых браузерах.  
  **Mitigation:** всегда `preventDefault()` в обработчике активации.

- **[Trade-off]** `id` остаётся `btn-brand` при семантике ссылки — лёгкая когнитивная несогласованность для разработчиков, зато без массовых правок.
