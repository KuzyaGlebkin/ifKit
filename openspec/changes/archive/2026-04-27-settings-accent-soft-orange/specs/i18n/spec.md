## ADDED Requirements

### Requirement: Стабильный ключ UI для пресета акцента `orange`

Движок SHALL определять стабильный ключ системного UI (в пространстве, согласованном с `ui-keys` / `define-game-api`, например `ui.settingsAccentOrange`) для подписи сегмента пресета `orange` в ряду «Акцент» модалки настроек, с **встроенным** fallback в `builtin-ui` (или эквивалент) **как минимум** для русского и английского, согласованно с `u(key)`. Ключ, соответствующий удалённому пресету `violet` (`ui.settingsAccentViolet`), SHALL NOT оставаться единственным идентификатором для оранжевого пресета.

#### Scenario: `u` возвращает подпись для оранжевого пресета

- **WHEN** вызывается `u` для ключа подписи пресета `orange` и активна локаль с заполненным значением
- **THEN** возвращается непустая строка, семантически соответствующая оранжевому/мягко-оранжевому пресету акцента
