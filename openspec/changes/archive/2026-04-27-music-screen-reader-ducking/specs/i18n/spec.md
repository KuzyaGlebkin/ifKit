## ADDED Requirements

### Requirement: Системные ключи для тихой музыки со скринридерами

Движок SHALL определять стабильные ключи системного UI в пространстве `u()` для подписи переключателя `quietMusicForScreenReader` (и при необходимости отдельный ключ краткого пояснения для вспомогательных технологий). Для каждого ключа SHALL быть встроенный fallback в словаре движка (`builtin-ui` или эквивалент) **как минимум** для русского и английского, согласованный с `u(key)` и `settings-modal`.

#### Scenario: Ключ подписи переключателя разрешён через u

- **WHEN** вызывается `u` для ключа подписи переключателя тихой музыки для скринридера
- **THEN** возвращается непустая строка (из локали автора или встроенного fallback)
