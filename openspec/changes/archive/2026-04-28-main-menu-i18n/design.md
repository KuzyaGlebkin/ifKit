## Context

Стартовое меню заполняется в `initSessionMainMenu`: заголовок и описание задаются из `config.meta` один раз как `textContent`. Кнопки и aria-labels обновляются в `refreshSessionMainMenuChrome()` через `u()`. Обработчик `setLocaleChangeHandler` уже вызывает `refreshSessionMainMenuChrome` при смене языка, но DOM заголовка и описания туда не входят. Игровой текст для сцен использует `` t`…` `` с ключами — строками на языке источника в `*.game.json`.

## Goals / Non-Goals

**Goals:**

- Заголовок и описание главного меню разрешать через тот же слой `_game[_resolvedLang]`, что и `t`, без дублирования семантики ключей.
- Обновлять эти два блока при каждом вызове `refreshSessionMainMenuChrome` (уже связанном со сменой языка).

**Non-Goals:**

- Отдельные UI-ключи `UiKey` для title/description (избыточно, если ключи совпадают с игровыми фразами из `meta`).
- Изменение формата `defineGame({ meta })` (поля остаются строками TypeScript).

## Decisions

1. **Разрешение строки без tagged template** — добавить в `i18n` экспорт функции вида `translateGameString(sourceKey: string): string`, реализованную тем же поиском `_game[_resolvedLang]?.[sourceKey]` и fallback на `sourceKey`, что и неинтерполируемый `t`. Это избегает хранения `TemplateStringsArray` в движке и держит один источник правды.

2. **Где применять** — `session-main-menu` сохраняет «сырой» `title` и `description` из опций (как сейчас), но при установке текста в DOM и в `refreshSessionMainMenuChrome` использует `translateGameString`. Пустое описание по-прежнему скрывает блок (`hidden`).

3. **Образцовая игра** — дополнить `src/locales/en.game.json` парами для точных строк из `meta` в `game.ts`, чтобы smoke-тест на английском показывал перевод.

**Альтернатива:** вызывать `` t`…` `` только если автор оборачивает литералы в `game.ts` — отклонено: до инициализации i18n порядок выполнения модулей делает это хрупким; ключи как литералы в `meta` надёжнее.

## Risks / Trade-offs

- **[Risk]** Автор забыл добавить ключ в `*.game.json` → видна исходная строка (как для обычных строк сцен).  
  **Mitigation:** то же поведение, что у `t`; документировано в delta `define-game-api`.

- **[Risk]** Очень длинные описания в `meta` как ключи JSON — неудобны для diff.  
  **Mitigation:** приёмлемый компромисс проекта; при желании позже можно ввести короткие id в `meta` отдельным изменением.

## Migration Plan

Изменение обратно совместимо: без записей в локалях отображение совпадает с текущим (fallback на исходный текст).
