## ADDED Requirements

### Requirement: Подписи кнопок панели #controls

Текстовые подписи (`.ifk-btn-label`) на кнопках `#btn-undo`, `#btn-redo`, `#btn-saves`, `#btn-settings` SHALL **не** быть **фиксированы** в HTML одним языком: движок SHALL выставлять их из системного словаря (например `u(ключ)`) **после** инициализации i18n **и** обновлять при смене resolved языка (в т.ч. из настроек), чтобы они совпадали с активной локалью. Иконки Lucide остаются **декоративными** (`aria-hidden`); **видимая** подпись несёт язык для зрячих.

#### Scenario: Смена языка в настройках обновляет панель

- **WHEN** игрок сменил язык на `en` **и** для `en` (или встроенного fallback) заданы системные строки для подписей панели
- **THEN** подписи «Назад» / «Вперёд» / «Сохранения» / «Настройки» (или эквиваленты) **отображаются** на `en` **без** перезагрузки страницы

#### Scenario: Старт

- **WHEN** игра **завершила** инициализацию `defineGame` и i18n
- **THEN** панель **не** остаётся **навсегда** в языке, зафиксированном **только** в `index.html`, если resolved язык **другой** и есть строки для `u`
