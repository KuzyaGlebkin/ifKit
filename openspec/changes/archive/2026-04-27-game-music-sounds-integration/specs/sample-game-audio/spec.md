## ADDED Requirements

### Requirement: Фоновая музыка демо-игры
Демо-игра в `src/scenes.ts` SHALL импортировать `Adventure.mp3` из `src/assets/` через Vite и вызывать `PlayMusic` с этим URL в теле каждой сцены, где остаётся фоновое музыкальное сопровождение (как минимум сцены `start` и `clearing`), с одним и тем же значением `src`, чтобы существующая логика намерения не останавливала трек при смене сцены.

#### Scenario: Музыка задана в обеих локациях
- **WHEN** игрок на поляне или в лесу и сцена отрисована
- **THEN** в теле сцены выполнен вызов `PlayMusic` с URL, полученным из `src/assets/Adventure.mp3`

#### Scenario: Тот же src при навигации
- **WHEN** игрок переходит между `start` и `clearing`
- **THEN** в обеих сценах передаётся идентичный аргумент в `PlayMusic` (тот же импорт)

---

### Requirement: Звук подбора монеты
Демо-игра SHALL при успешном действии «Подобрать монету» вызывать `Sound` с URL из `src/assets/impactGeneric_light_001.ogg` (Vite-импорт) в колбэке `act` этого действия.

#### Scenario: Монета и звук
- **WHEN** пользователь активирует действие подбора монеты
- **THEN** воспроизводится однократный звук из `impactGeneric_light_001.ogg` и обновляется состояние золота

---

### Requirement: Звук подбора ветки
Демо-игра SHALL при успешном действии «Поднять ветку» вызывать `Sound` с URL из `src/assets/impactWood_heavy_001.ogg` (Vite-импорт) в колбэке `act` этого действия.

#### Scenario: Ветка и звук
- **WHEN** пользователь активирует действие поднятия ветки
- **THEN** воспроизводится однократный звук из `impactWood_heavy_001.ogg` и обновляется флаг `hasBranch`

---

### Requirement: Согласованность с API движка
Реализация SHALL использовать экспортированные из `./ifKit` (или `ifKit` по соглашению проекта) идентификаторы `PlayMusic` и `Sound` без обхода внутренних модулей движка; источник звука передаётся строкой URL, полученной от импорта ассета.

#### Scenario: Публичный API
- **WHEN** читается код `scenes.ts`
- **THEN** `PlayMusic` и `Sound` импортируются из пакета/алиаса игры, согласованного с остальными сценами
