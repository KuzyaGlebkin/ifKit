## MODIFIED Requirements

### Requirement: Открытие модалки по кнопке #btn-settings
Движок SHALL вешать обработчик на кнопку `#btn-settings` при инициализации. По клику SHALL открываться модальное окно настроек.

#### Scenario: Клик на кнопку «Настройки» открывает модалку
- **WHEN** пользователь нажимает кнопку «Настройки» в `#controls`
- **THEN** модальное окно появляется

#### Scenario: Модалка не открывается повторно если уже открыта
- **WHEN** пользователь нажимает «Настройки» при уже открытой модалке
- **THEN** повторного открытия не происходит

### Requirement: Закрытие модалки
Модалка SHALL закрываться по: клику на кнопку ✕, нажатию `Escape`, клику на backdrop (область вне окна).

#### Scenario: Закрытие по Escape
- **WHEN** модалка открыта и пользователь нажимает `Escape`
- **THEN** модалка закрывается

#### Scenario: Закрытие по клику на backdrop
- **WHEN** пользователь кликает мимо окна на затемнённый фон
- **THEN** модалка закрывается

## REMOVED Requirements

### Requirement: Доступность модалки (ARIA)
**Reason**: Намеренный откат a11y-контракта; см. change `remove-screen-reader`, design.md.
**Migration**: Снять `role="dialog"`, `aria-modal`, `aria-labelledby` и логику focus trap из разметки и скрипта; поведение открытия/закрытия — по оставшимся сценариям.
