## ADDED Requirements

### Requirement: Системные ключи для объявляемой громкости ползунков
Движок SHALL определять стабильные ключи `UI` для строки, **озвучиваемой** вспомогательными технологиями для текущего уровня громкости (музыка и звуки), с **плейсхолдером** для целого числа 0–100, чтобы автор локали мог задать порядок «число + проценты» в соответствии с языком. Для каждого ключа (или одного общего ключа, если движок использует единый шаблон) SHALL быть встроенный fallback **как минимум** для русского и английского в `builtin-ui` (или эквивалент), согласованный с `u(key)`.

#### Scenario: Ключ для музыки разрешён
- **WHEN** вызывается `u` для ключа объявляемой громкости музыки с подставленным числом
- **THEN** возвращается непустая локализованная строка, содержащая **целое** процентное значение, а не сырую десятичную долю 0.0–1.0

#### Scenario: Ключ для звуков разрешён
- **WHEN** вызывается `u` для ключа объявляемой громкости звуков с подставленным числом
- **THEN** возвращается непустая локализованная строка, содержащая **целое** процентное значение, а не сырую десятичную долю 0.0–1.0
