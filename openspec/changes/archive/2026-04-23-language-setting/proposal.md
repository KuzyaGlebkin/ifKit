## Why

Игроки должны иметь возможность переключать язык игры прямо из модального окна настроек. Без явного UI пользователь не может воспользоваться многоязычностью, даже если автор передал несколько локалей через `defineGame()`.

## What Changes

- В модальном окне настроек появляется секция «Язык» с группой кнопок — по одной на каждый доступный язык.
- Секция видна только когда `locales` содержит более одного языка; при одном языке или без `locales` — скрыта.
- Нажатие на кнопку языка применяет его немедленно, сохраняет в хранилище и перерисовывает текущую сцену.
- Активный язык визуально выделен среди кнопок.
- Сброс настроек возвращает язык к значению `authorDefaults.language`.
