## ADDED Requirements

### Requirement: `LocaleBundle` в шаблоне с импортируемыми `*.game.json` и `*.ui.json`

Когда в шаблоне проекта для кода `code` используется отдельный `src/locales/<code>.game.json`, шаблон SHALL также предусматривать `src/locales/<code>.ui.json` и передаву `code: { game: <импорт game.json>, ui: <импорт ui.json> }` в `defineGame({ locales: { … } })` для **всех** таких `code` в **рекомендуемой** конфигурации. Плоский формат `locales: { code: { … } }` (только `t`) в спецификации `define-game-api` остаётся допустимым **без** изменения норматива, но **образцовая** игра ifKit SHALL демонстрировать пару JSON, если в ней больше одного языка или требуется неанглийский chrome.

#### Scenario: Согласованные импорты в `game.ts`

- **WHEN** автор смотрит шаблон `game.ts` с двумя языками
- **THEN** в нём **видны** четыре импорта (два `*.game.json` и два `*.ui.json`) **или** эквивалентная структура, явно сопоставляющая `game` и `ui` на каждый `code`
