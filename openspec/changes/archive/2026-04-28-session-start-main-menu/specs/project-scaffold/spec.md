## ADDED Requirements

### Requirement: Каркас страницы и слой стартового меню

Минимальный `index.html` SHALL допускать размещение слоя стартового меню (контейнер с стабильным `id`, предназначенный под полноэкранный UI), который движок может показывать и скрывать без нарушения наличия `#controls` и `#content` в DOM. При видимости стартового меню навбар SHALL не отображаться в соответствии со спецификацией `session-main-menu`.

#### Scenario: Контейнер меню существует

- **WHEN** страница загружена
- **THEN** разметка содержит объявленный корневой элемент для слоя меню или движок создаёт его при инициализации без дублирования конфликтующих `id`
