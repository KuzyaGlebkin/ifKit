## ADDED Requirements

### Requirement: UI-ключи стартового меню и логотипа

Перечисление `UiKey` SHALL быть расширено стабильными ключами для: кнопки «Новая игра», «Загрузить» (в контексте меню, если отличается от тулбара), подписи интерактивного логотипа/главного меню, при необходимости — заголовка экрана меню. В `BUILTIN_UI` и шаблоне `template.ui.json` SHALL присутствовать согласованное заполнение этих ключей для языка оригинала и путей перевода.

#### Scenario: Ключи присутствуют в ui-keys

- **WHEN** разработчик открывает `src/ifKit/ui-keys.ts`
- **THEN** перечислены идентификаторы для строк стартового меню, используемые через `u()`
