## ADDED Requirements

### Requirement: Интерактивный логотип для входа в стартовое меню

Помимо требований к визуальному логотипу в `#controls`, движок SHALL предоставлять интерактивный контрол (кнопка или ссылка с ролью `button`), оборачивающий логотип или заменяющий его для pointer/клавиатуры, который по активации переводит приложение в стартовое меню сессии согласно спецификации `session-main-menu` (включая автосохранение и остановку музыки).

#### Scenario: Активация логотипа

- **WHEN** игрок активирует интерактивный логотип кликом или клавишей активации
- **THEN** выполняется сценарий «Клик по логотипу в навбаре» из `session-main-menu`
