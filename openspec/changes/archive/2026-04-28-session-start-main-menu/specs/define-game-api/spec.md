## ADDED Requirements

### Requirement: Поле meta — название и описание для стартового меню

`defineGame()` SHALL принимать необязательное поле `meta` с объектом минимум с полем **название игры** (`title`), отображаемым в стартовом меню. Поле **описание** (`description` или эквивалент) SHALL быть опциональным текстом автора под заголовком. Значения SHALL быть строками в терминах TypeScript; локализация MAY достигаться передачей строк, уже обработанных через `t`, или через ключи/шаблоны по согласованности с `i18n` и авторскими локалями.

#### Scenario: Передано title

- **WHEN** автор вызывает `defineGame({ state, scenes, meta: { title: 'Моя история', description: '…' } })`
- **THEN** стартовое меню показывает переданное название и описание в соответствующих блоках

#### Scenario: Отсутствие meta

- **WHEN** поле `meta` не передано
- **THEN** движок SHALL использовать задокументированный fallback (например имя пакета или нейтральная строка) для заголовка без аварийного завершения
