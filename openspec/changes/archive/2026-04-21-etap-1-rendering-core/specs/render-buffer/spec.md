## ADDED Requirements

### Requirement: Строковый буфер накапливает HTML-фрагменты
Движок SHALL поддерживать глобальный строковый буфер для накопления HTML-строк в процессе выполнения функции сцены.

#### Scenario: Запись в буфер через теговую функцию
- **WHEN** теговая функция `P('Текст')` вызывается внутри функции сцены
- **THEN** строка `<p>Текст</p>` конкатенируется к содержимому буфера

#### Scenario: Несколько теговых функций накапливаются в порядке вызова
- **WHEN** в функции сцены последовательно вызываются `H1('Заголовок')` и `P('Абзац')`
- **THEN** буфер содержит `<h1>Заголовок</h1><p>Абзац</p>` в том же порядке

### Requirement: Буфер очищается перед каждым вызовом сцены
Движок SHALL сбрасывать буфер до пустой строки перед вызовом любой функции сцены.

#### Scenario: Буфер пуст в начале первой сцены
- **WHEN** игра запускается и вызывается первая сцена
- **THEN** буфер не содержит посторонних данных

#### Scenario: Буфер пуст в начале следующей сцены
- **WHEN** пользователь переходит к новой сцене через `goto()`
- **THEN** HTML от предыдущей сцены не присутствует в буфере до вызова новой сцены

### Requirement: Буфер сбрасывается в DOM после выполнения сцены
Движок SHALL применять содержимое буфера к целевому DOM-элементу сразу после завершения функции сцены.

#### Scenario: Содержимое сцены появляется в DOM
- **WHEN** функция сцены завершила выполнение
- **THEN** свойство `innerHTML` целевого элемента (`#scene`) равно накопленному буферу

#### Scenario: Пустой буфер очищает целевой элемент
- **WHEN** функция сцены не вызывает ни одной теговой функции
- **THEN** целевой элемент очищается (`innerHTML = ''`)

### Requirement: Буфер доступен для инспекции в тестах
Движок SHALL экспортировать функции `clearBuffer()` и `getBuffer()` из модуля `renderer.ts`.

#### Scenario: getBuffer возвращает текущее содержимое
- **WHEN** вызывается `getBuffer()` после `P('Тест')`
- **THEN** возвращается строка, содержащая `<p>Тест</p>`

#### Scenario: clearBuffer сбрасывает буфер
- **WHEN** вызывается `clearBuffer()` после записи в буфер
- **THEN** `getBuffer()` возвращает пустую строку
