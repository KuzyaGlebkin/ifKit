## ADDED Requirements

### Requirement: Императивные теговые функции пишут в буфер
Движок SHALL экспортировать набор теговых функций в верхнем регистре (`P`, `H1`, `H2`, `H3`, `Div`, `Span`, `Img`, `Hr`, `Br`), каждая из которых добавляет соответствующий HTML-тег в глобальный буфер рендеринга.

#### Scenario: P добавляет параграф в буфер
- **WHEN** автор вызывает `P('Ты в лесу.')`
- **THEN** в буфер добавляется строка `<p>Ты в лесу.</p>`

#### Scenario: H1 добавляет заголовок первого уровня в буфер
- **WHEN** автор вызывает `H1('Лес')`
- **THEN** в буфер добавляется строка `<h1>Лес</h1>`

#### Scenario: Img добавляет изображение в буфер
- **WHEN** автор вызывает `Img('assets/images/forest.png', 'Лес')`
- **THEN** в буфер добавляется строка `<img src="assets/images/forest.png" alt="Лес">`

#### Scenario: Hr добавляет горизонтальную линию без содержимого
- **WHEN** автор вызывает `Hr()`
- **THEN** в буфер добавляется строка `<hr>`

### Requirement: Функциональные теговые функции возвращают HTML-строку
Движок SHALL экспортировать набор теговых функций в нижнем регистре (`p`, `h1`, `h2`, `h3`, `div`, `span`, `img`, `hr`, `br`), каждая из которых возвращает HTML-строку, не изменяя буфер.

#### Scenario: p возвращает строку без записи в буфер
- **WHEN** автор вызывает `const html = p('Текст')` и при этом не происходит других операций
- **THEN** `html` равно `<p>Текст</p>` и буфер остаётся без изменений

#### Scenario: Функциональные функции допускают композицию
- **WHEN** автор вызывает `P(b('Жирный') + ' обычный')`
- **THEN** в буфер добавляется строка содержащая `<b>Жирный</b> обычный` внутри тега `<p>`

### Requirement: Теговые функции принимают атрибуты через необязательный объект
Движок SHALL позволять передавать HTML-атрибуты в теговые функции через необязательный объект `attrs`.

#### Scenario: Передача CSS-класса через атрибуты
- **WHEN** автор вызывает `P('Текст', { class: 'highlight' })`
- **THEN** в буфер добавляется строка `<p class="highlight">Текст</p>`

#### Scenario: Вызов без атрибутов работает без ошибок
- **WHEN** автор вызывает `P('Текст')` без второго аргумента
- **THEN** в буфер добавляется строка `<p>Текст</p>` без лишних атрибутов

### Requirement: Поддерживаемый набор тегов первого уровня фиксирован
Движок SHALL поддерживать ровно следующие теги в Этапе 1: `p`, `h1`, `h2`, `h3`, `div`, `span`, `img`, `hr`, `br`, `b`, `i`, `a`. Каждый тег доступен в обоих регистрах (кроме `hr` и `br`, которые не имеют содержимого).

#### Scenario: Все перечисленные функции экспортируются из ifKit
- **WHEN** автор пишет `import { P, H1, p, h1 } from './src/ifKit'`
- **THEN** TypeScript не выдаёт ошибки об отсутствующих экспортах
