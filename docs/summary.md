# Документация ifKit — Оглавление

## 1. Введение

- Что такое ifKit?
- Для кого этот проект (от новичка до публикации в Steam)
- Основные возможности

## 2. Начало работы

- Установка и системные требования
- Первая игра за 5 минут
- Структура проекта (`game.ts` и остальное)

## 3. Основные концепции

- Сцены и переходы
- Состояние игры (глобальные и локальные переменные)
- Цикл рендеринга (буфер, теговые функции)

## 4. API — Справочник

- Теговые функции разметки (`P`, `p` и другие)
- Определение сцен (словарь `Record<string, () => void>`)
- Действия и переходы (type-safe ключи сцен)
- Управление глобальным состоянием
- Управление локальным состоянием (вспомогательная функция)
- Звуковое API (фоновая музыка, звуковые эффекты)

## 5. Интерфейс пользователя

- Зоны интерфейса (постоянные элементы, сцена, действия, переходы)
- Система переходов (форма с радиокнопками)
- Уведомления об изменении переменных (snackbar)
- Настройки (тема, размер шрифта, громкость)

## 6. Система сохранений

- Слоты сохранений (номер, дата, описание, скриншот)
- История состояний (последние N шагов, откат назад)
- Автосохранение

## 7. Интернационализация

- Как устроена автоматическая i18n
- Добавление нового языка

## 8. Сборка и развёртывание

- Сборка в один HTML-файл (`vite-plugin-singlefile`)
- Desktop-приложение (Tauri)
- Mobile-приложение (сборка на Android-смартфоне)

## 9. Расширение и кастомизация

- Архитектура плагинов / хуков
- Создание собственных теговых функций
- Пример кастомной темы

## 10. Доступность и адаптивность

- Поддержка скринридеров
- Адаптивный дизайн (разные размеры экрана)

## 11. Продвинутые темы

- Публикация в Steam
- Оптимизация производительности
- Интеграция с внешними сервисами
